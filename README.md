# 傲視系統優化大師 (Vision System Optimizer)

一款基於 Python 與 PySide6 打造的專業級系統優化工具。結合極致的現代化 UI 設計與強大的系統優化核心，提供一站式的電腦維護體驗。

## ✨ 主要功能

- **🚀 一鍵加速 (One-Tap Boost)**
  - 智慧分析並釋放系統記憶體 (RAM)。
  - 清理後台冗餘進程，瞬間提升電腦回應速度。

- **🧹 垃圾清理 (Junk Cleaner)**
  - 深度掃描系統暫存檔、應用程式快取。
  - 安全清理無用檔案，釋放寶貴的磁碟空間。
  - 即時顯示掃描結果與預計釋放空間。

- **📊 即時儀表板 (Live Dashboard)**
  - 銀河黑深色主題儀表板。
  - 環形進度條動態展示 CPU、RAM 與磁碟使用率。
  - 磨砂玻璃 (Glassmorphism) 視覺效果。

- **⚡ 啟動項管理 (Startup Manager)**
  - 檢視所有開機自動啟動的程式與路徑。
  - 協助您分析拖慢開機速度的原因。

## 🛠️ 安裝與執行

### 環境需求
- Python 3.10+
- Windows 10/11

### 快速開始

1. 安裝依賴套件：
   ```bash
   pip install -r requirements.txt
   ```

2. 啟動程式：
   ```bash
   python main.py
   ```
